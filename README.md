AI Sign Language Recognition System – README
📌 Overview
The AI Sign Language Recognition System is a real-time computer vision application that detects and recognizes sign language gestures and converts them into text. The project aims to improve accessibility for people with hearing and speech impairments using deep learning and AI.
🚀 Features
Real-time hand gesture detection using webcam
Sign language to text conversion
Deep learning-based gesture recognition
High accuracy prediction model
User-friendly interface
Real-time frame processing
